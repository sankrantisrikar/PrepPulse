#### "Failed to transcribe audio" with ElevenLabs

**Solutions:**
1. Verify ElevenLabs API key is valid
2. Check audio format (webm, mp3, wav supported)
3. Ensure audio file is not corrupted
4. Check ElevenLabs account has sufficient credits
5. Verify rate limits not exceeded (10 req/sec)

#### Comparing transcription quality

If you want to compare ElevenLabs STT with your previous Whisper setup:
1. Record the same answer twice
2. Compare transcription accuracy
3. ElevenLabs should be equal or better, especially for ElevenLabs-generated voice
